# Receiver App

The Receiver App is a Qt-based C++ application responsible for receiving incoming data and handling communication logic within the system.

This application acts as the entry point for data into the overall Receiver & Store Tracker System.

## Files Overview

| File | Description |
|-----|------------|
| `ReceiverApp.pro` | Qt project configuration file |
| `recv_main.cpp` | Application entry point |
| `receivermainwindow.h` | Main window class definition |
| `receivermainwindow.cpp` | Main window implementation |
| `receivermainwindow.ui` | UI layout designed with Qt Designer |

## Build Instructions

1. Open `ReceiverApp.pro` in Qt Creator
2. Configure the kit (Desktop)
3. Build and run the project

No additional dependencies are required beyond Qt.
