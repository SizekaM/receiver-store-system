# Receiver & Store Tracker System

This repository contains two Qt-based C++ applications that work together as a single system.

The system is designed to receive data through the Receiver App and manage, track, or visualize that data through the Store Tracker App.

## Repository Structure


## Applications

### Receiver App
Responsible for receiving incoming data, handling communication logic, and preparing data for storage or further processing.

### Store Tracker App
Responsible for tracking, managing, and displaying data received from the Receiver App.

## Technology Stack
- C++
- Qt Widgets
- Qt Designer (.ui files)

Each application builds independently using its own `.pro` file and can be opened separately in Qt Creator.
